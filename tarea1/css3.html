Aplicar CSS3


Elemento Descripción Ejemplo
colors CSS3 establece la notación RGBA. Los tres primeros son los bien conocidos canales RGB (rojo, verde y azul) y el cuarto parámetro es el canal Alpha, representa el grado de transparencia u opacidad del color. 0 totalmente transparente y 1 totalmente opaco. section {
color: rgba(151, 117, 43, 0.5);
}
backgrounds Gracias a la propiedad background se puede establecer de forma directa el color de fondo (background-color), la imagen de fondo (background-image), su posición (background-position), si la imagen es fija o no (background-attachment) y/o si la imagen se repite o no (background-repeat).
Fuente: https://uniwebsidad.com/libros/referencia-css2/background 		Por ejemplo para dar solo color de fondo a un elemento:

footer {
background: rgba(151, 117, 43, 0.5);
}
borders Para establecer simultáneamente el grosor, estilo y color del borde de un elemento.
Fuente: https://uniwebsidad.com/libros/referencia-css2 		Por ejemplo para dar solo color de fondo a un elemento:

div {
border: solid rgba(45, 147, 50, 0.7);
}
margins Para establecer simultáneamente el grosor, estilo y color del borde de un elemento.
Fuente: https://uniwebsidad.com/libros/referencia-css2 		Por ejemplo para dar solo color de fondo a un elemento:

div {
border: solid rgba(45, 147, 50, 0.7);
}
texts Permite aplicar diferentes propiedades text-align, text-transform (mayúscula y minúscula), color, text-decoration, text-indent (espaciado), text-shadow.
Fuente: https://www.w3schools.com/css/css_text.asp 		Por ejemplo:

p {
text-indent: 50px;
}
fonts Esta propiedad permite establecer el tipo de letra o fuente a utilizar: font-family, font-style, font-size (tamñano-fuente), font-weight.
Por ejemplo:

p {
font-style: italic;
} Por ejemplo:

p {
font-size: 40px;
}
height - width Estas propiedades se utilizan para establecer la altura y el ancho de un elemento.
Por ejemplo:

div class="icono" {
width: 25%;
height: 50px;
}
links Existen cuatro estados para la propiedad "link":
a:link - link sin visitar.
a:visited - link visitado.
a:hover - link activado cuando el usuario pasa el mouse sobre el elemento.
a:active - se activa el link cuando el objeto es clickeado.
/*link sin visitar */
a:link {
color: rgba(23,123, 0,34, 03);
}

/* link visitado*/
a:visited {
color: blue;
}

/* link activado con el mouse */
a:hover {
color: rgba(67, 198, 220,34, 08);
}

/* link seleccionado*/
a:active {
color: rgba(23,123, 0,34, 03);
}

lists La propiedad list-style-type especifica el tipo de marcador de elemento de lista. Ejemplo de los marcadores para elementos de lista disponibles:

ul.a {
list-style-type: circle;
}
Coloca viñeta de circulo.

ul.b {
list-style-type: square;
}
Coloca viñeta de un cuadrado.

ol.c {
list-style-type: upper-roman;
}
Ej: I. Jugo.
II. Sopa.

ol.d {
list-style-type: lower-alpha;
}
Ej: 1. Jugo.
2. Sopa.
position Esta propiedad especifica el tipo de método de posicionamiento utilizado para un elemento.
Existen cinco valores:
- static
- relative
- fixed
- absolute
- sticky
div.fixed {
position: fixed;
bottom: 0;
right: 0;
width: 300px;
border: 3px solid rgba(123, 0,0, 1); }
overflow La propiedad controla lo que sucede con el contenido que es demasiado grande para caber en un área.
Tiene los siguientes valores:
visible - Predeterminado. El desbordamiento no se recorta. El contenido se muestra fuera de la caja del elemento.
oculto: el desbordamiento se recorta y el resto del contenido será invisible.
scroll: el desbordamiento se recorta y se agrega una barra de desplazamiento para ver el resto del contenido.
auto: similar al desplazamiento, pero agrega barras de desplazamiento solo cuando es necesario.
div {
overflow: auto;
}
float La propiedad se utiliza para posicionar y dar formato al contenido. Permite que una imagen flote en determinado lugar del texto en un contenedor.
left - El elemento flota a la izquierda de su contenedor
right - El elemento flota a la derecha de su contenedor
none: el elemento no flota (se mostrará justo donde aparece en el texto). esto es por defecto
heredar: el elemento hereda el valor flotante de su padre.
img {
float: right;
}
