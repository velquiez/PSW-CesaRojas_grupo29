Uso etiquetas HTML5


Etiqueta Descripción Sintaxis
< header > Señala un grupo de elementos de introducción de una sección o documento.
Normalmente se usa para incluir los encabezados ( <h1> – <h6> o <hgroup> ) 		<header>
  <h1> Aquí agregas el título </h1>
</header>
< nav > Esta etiqueta representa el menú de navegación de la página. Por lo general, va a acompañada por las etiquetas < ul > y < li >. <nav>
  <ul>
    < li> <a href="#"> Principal </a> </li>
    < li> <a href="#"> Tabla html5 </a> </li>
    < li> <a href="#"> Tabla CSS3 </a> </li>
    < li> <a href="#"> Universidad </a> </li>
  </ul>
</nav>
<p> Define una parte que debe mostrarse como un párrafo. <p>
  Aquí va el contenido del párrafo...etc.
</p>
< article > Declara un contenido que es independiente de cualquier otras secciones. < article>
  Agrega el texto para blog, periódico, etc.
</ article>
< h1 >
< h2 >
< h3 > ...
< h6 >		 Sirve para implementar seis niveles de encabezado del documento, < h1> es el más importante, y < h6> , el menos importante. < h1>
  Escribe el título principal.
</ h1>
< aside > Representa una sección de una página que consiste en contenido que está indirectamente relacionado con el contenido principal del documento. < aside>
  Por ejemplo, coloca en esta sección publicidad a tu página.
</ aside>
< figure > De uso específico para insertar figuras o imágenes, con sus atributos. < figure>
  < img class="img-fluid" src="img/css3.png" alt="logo css3">.
</ figure>
< footer > De uso específico para insertar figuras o imágenes, con sus atributos. < footer>

< section > Representa una sección genérica de un documento. Sirve para determinar qué contenido corresponde a qué parte de un esquema. < section>
  Agrega contenido diverso.
</ section>
< table > De uso específico para generar tabla. Va acompañado de < tr > para filas y < td > para las columnas. < table>
  < tr>
   < th> Nombres < /th>
   < th> Apellidos < /th>
  </tr>
  <tr>
   < td> Augusto </td>
   < td> Esquivel </td>
  </tr>
</ table>
